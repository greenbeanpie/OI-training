# 统计方案

## 题目描述

给你一个 $n \times m$ 的矩阵 $a$，问有多少种从$(1,1)$到 $(n,m)$ 只能往下走或者往右走的路径，使得路径上的权值的异或和为一个给定的数 $k$。

## 输入格式

第一行三个整数 $n$,$m$,$k$。

接下来 $n$ 行，每行 $m$ 个整数表示 $a_{i,j}$。

## 输出格式

一个整数，表示合法的路径数。

## 样例 #1

### 样例输入 #1

```
3 3 11
2 1 5
7 10 0
12 6 4
```

### 样例输出 #1

```
3
```

## 提示

$1 \le n,m \le 20$

$0 \le k \le 10^{18}$

$0 \le ai,j \le 10^{18}$
